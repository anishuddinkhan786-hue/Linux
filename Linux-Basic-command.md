# How to check system date and time 

```bash
    anish@ANISH:~$ date
    Sun Sep 28 12:35:05 UTC 2025
    anish@ANISH:~$ date +%D
    09/28/25
    anish@ANISH:~$ date +%T
    12:35:24
    anish@ANISH:~$ date +%H:%M:%S
    12:35:52
    anish@ANISH:~$ date +%dd:%mm:%yyyy
    28d:09m:25yyy
    anish@ANISH:~$ date +%d:%m:%y
    28:09:25
```
